Floating Point to Hex Converter

Here are the Python files that are needed to make your own:

  • floattohexmodule.c (this is the C file that compiles into a Python module)
  • floattohex.cgi

About this project

I started looking for one of these becausse I thought it would be useful for work (turned out I didn't need it). I found a Java applet version, and thought it would be nice to make a lighter-weight solution. I might add support for endianness, too.
